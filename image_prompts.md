## Usage notes for agents

- Generate in order: logo → menu_bg → portraits (reuse same style description) → stone icons → locations → boards → results → UI.
- Keep character and stone style consistent by appending the same style phrase to each prompt in that section.
- Save outputs into `game/images/` (or your project’s image folder) with the filenames above.
- Ren'Py will look for images in `game/images/` by default; use the filename without path in script/screens (e.g. `"portrait_eevee"`).
